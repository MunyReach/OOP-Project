**Project Title:**

" OOP Bookstore "


**Project Type:**
Bookstore Management Project 


**Team Members:**

Chap Chanliya, Munyreach Sakal, Sereywath Rathanak, Porleak Vitou


**Project Description:**

We are creating a bookstore management system that has system for staff to manage customers, books and sales using OOP concepts in Java through command-line interface and planning to implement GUI if plausible. 


**Project Features:**

- Core features that our project will include: 
 
+ Book Management: Staff should be able to add, update, delete books and manage stock.
+ Customer Management: Add, update or delete with information of customers.
+ Sales/Order Management: create, view, calculate order and generate receipts.
+ Staff Management: Login as Manager or Cashier. 

- Key concepts from OOP that we will implement:

+ Encapsulation
+ Inheritance
+ Abstract Class
+ Interface
+ Polymorphism
